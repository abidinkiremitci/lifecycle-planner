# Equipment & Gym Space

## Gym Dimensions

- **Width:** 5 meters
- **Length:** 4 meters
- **Total area:** 20 m²
- **Note:** Once equipment is set up, there is just enough space for movement — no extra room. Plan workouts accordingly (no long walking lunges, limit carries to short shuttles).

## Available Equipment

| Equipment | Notes |
|---|---|
| Barbell + plates | Olympic barbell with bumper/iron plates |
| Squat rack | Full-size rack |
| Dumbbells | Adjustable or fixed set |
| Kettlebells | Various weights |
| Bench | Flat/incline |
| Plyo box | For box jumps, step-ups, seated box jumps |
| Concept2 rower | Indoor cardio option |
| Rings | Gymnastics rings (ceiling or rack mounted) |
| Dip bars | Parallel bars for dips and support work |
| Pull-up bar | Pull-up bar attached to squat rack. It's not adjusted to groud, so it's not suitable for kipping |
| Resistance bands | Resistance bands with different sizes |

## Not Available

- Assault/Echo bike
- Ski erg
- GHD machine
- Climbing rope
- Sled / prowler
- Cable machine

## Space Constraints for Programming

- **Max continuous carry/walk distance:** ~5 m (use 5 × 10 m shuttles for longer prescribed distances).
- **No single-distance prescriptions > 20 m.** Convert to in-place movements, shuttles, or timed holds.
- **Running:** Must leave the gym space. Always provide an indoor alternative (row or jump rope).
- **Equipment transitions:** Keep setups minimal — ideally ≤ 2 stations per workout to avoid crowding.
